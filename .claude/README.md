# Claude Code Configuration for Jarvis Voice System

This directory contains Claude Code configurations for enhanced development workflows.

## Structure

- `agents/` - Specialized subagents for delegation
- `skills/` - Workflow definitions and domain knowledge
- `commands/` - Slash commands for quick execution
- `rules/` - Always-follow guidelines
- `hooks/` - Trigger-based automations
- `contexts/` - Dynamic system prompt injection contexts

## Installation

The everything-claude-code plugin is configured via the `.claude-plugin/` directory.

To use these configurations in Claude Code:

1. Install the plugin (see `CLAUDE.md` in project root)
2. Or manually copy components to `~/.claude/` directories
